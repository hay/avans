Inlevereisen

  • Je code moet op Github staan.
  • Voortbouwen op andere code mag, copy-pasten en een beetje aanpassen niet.
  • Je moet kunnen uitleggen hoe je code werkt.
  • Je moet D3 gebruiken, maar je mag ook zoveel andere libraries gebruiken als je wilt.
  • Je code moet werken in Chrome en/of Firefox.
  • Je code hoeft niet te schalen naar andere schermbreedtes (responsive).
  • Je mag data hergebruiken uit het liefdeskaart project, maar dat hoeft niet.
  • Het is fijn als het er mooi uitziet, maar dat hoeft niet.
  • Vermeld bronnen bij je data.
